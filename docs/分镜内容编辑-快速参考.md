# 分镜内容编辑 - 快速参考

> 编辑动作描述和台词的快速指南

---

## 🎯 核心功能

**编辑分镜的动作描述和台词**
- ✏️ 修改动作描述（`actionSummary`）
- 💬 修改/添加/删除台词（`dialogue`）
- 💾 实时保存

---

## 📍 位置

**剧本清单** → **分镜列表** → **动作描述区域** → 悬停显示 ✏️

---

## ⚡ 快速操作

| 操作 | 步骤 | 说明 |
|------|------|------|
| 进入编辑 | 悬停动作描述 → 点击 ✏️ | 进入编辑模式 |
| 编辑动作 | 修改动作描述文本框 | 描述分镜的动作 |
| 编辑台词 | 修改台词文本框 | 添加/修改/删除台词 |
| 保存 | 点击 ✓ 保存 | 立即保存并退出 |
| 取消 | 点击 ✗ 取消 | 放弃修改 |

---

## 💡 为什么需要编辑？

### 常见问题
1. **AI描述太简单** - `Character walking` → 需要更多细节
2. **台词不符合角色** - `Hello.` → 需要更有个性
3. **描述不准确** - AI理解错误，生成了不相关的内容
4. **需要添加台词** - 分镜应该有台词但AI没有生成
5. **需要删除台词** - 纯动作镜头不需要台词

---

## 📝 编写技巧

### 动作描述模板
```
[主体] + [动作] + [情绪/细节] + [环境]

示例：
Alex walking confidently through the rain-soaked alley, 
shoulders back, eyes scanning the shadows
```

### 台词技巧

#### 符合角色性格
- **冷静型**：`I see. Proceed.`
- **热情型**：`Oh wow! This is amazing!`
- **神秘型**：`You'll understand... in time.`

#### 推动情节
- ✅ `The package arrives at midnight. Don't be late.`
- ❌ `Hello.`

#### 自然对话
- ✅ `You know what? I'm done.`
- ❌ `I am terminating my participation.`

## 🎬 实际案例

### 案例 1：增加细节

**原始**：
```
动作：Character walking
台词：Wow.
```

**优化**：
```
动作：Alex stops abruptly, eyes widening in disbelief as 
     he stares at the massive alien spacecraft hovering 
     silently above the desert
台词：This... this changes everything.
```

---

### 案例 2：删除台词

**原始**：
```
动作：Two people talking
台词：Goodbye.
```

**优化**：
```
动作：Sarah turns away, fighting back tears, while John 
     reaches out but stops himself, hand frozen in mid-air
台词：（删除 - 用沉默表达情感）
```

---

### 案例 3：添加台词

**原始**：
```
动作：Character looking at photo
台词：（无）
```

**优化**：
```
动作：Emma holding the faded photograph with trembling 
     hands, tears streaming down her face
台词：I miss you so much, Mom.
```

## ✅ 检查清单

编辑完成后，检查：

- [ ] 动作描述是否具体生动？
- [ ] 是否包含情绪和细节？
- [ ] 台词是否符合角色性格？
- [ ] 台词长度是否适合镜头时长？
- [ ] 是否需要同步更新画面提示词？

---

## ⚠️ 注意事项

### 1. 与画面提示词的关系
- 动作描述：给人看（中文，易读）
- 画面提示词：给AI看（英文，技术性）
- 修改动作描述后，建议检查画面提示词

### 2. 台词长度建议
- 短镜头（<5秒）：0-10字
- 中等镜头（5-10秒）：10-30字
- 长镜头（>10秒）：30-50字

### 3. 删除台词
- 清空台词文本框 = 删除台词
- 分镜变成纯动作镜头

---

## 🎯 最佳实践

### 动作描述
1. **具体而生动** - 包含动作、情绪、细节
2. **考虑镜头** - 描述镜头重点和运动
3. **保持简洁** - 100-200字符为宜

### 台词
1. **符合角色** - 体现角色性格和背景
2. **推动情节** - 传递关键信息
3. **自然流畅** - 避免生硬的书面语
4. **适当留白** - 有时无台词更有力量

## 💬 常见问题

**Q: 修改后画面提示词会自动更新吗？**  
A: 不会，需要手动检查和更新

**Q: 可以添加多段台词吗？**  
A: 当前每个分镜只支持一段台词

**Q: 台词支持哪些语言？**  
A: 支持任何语言

**Q: 删除台词后能恢复吗？**  
A: 可以立即重新编辑恢复
